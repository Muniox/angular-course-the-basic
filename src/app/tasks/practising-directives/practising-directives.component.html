Practising Directives

  1. Add a button which says 'Display Details'
  2. Add paragraph with any content of your choice (e.g. 'Secret Password = tuna')
  3. Toggle the display of that paragraph with the button created in the first step
  4. Log all button clicks in an array and output that array bellow the secret paragraph (maybe log a timestamp or simply an incrementing number)
  5. Starting at the 5th log item, give all future log items a blue background (via ngStyle) and white color (ngClass)
